# QA Copilot Web

Aplicación web para equipos QA/PO que permite:

- Cargar o pegar una historia de usuario o contrato API.
- Elegir modo de detección automática o tipo manual.
- Validar calidad con checklist y observaciones accionables.
- Generar casos de prueba cuando el insumo cumple validaciones.
- Exportar casos generados a CSV.

## Ejecutar localmente

```bash
python3 -m http.server 8080
```

Luego abrir: `http://localhost:8080`
